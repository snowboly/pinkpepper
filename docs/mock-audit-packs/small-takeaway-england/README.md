# Small Takeaway England Mock Audit Pack

This pack is designed for `Virtual Audit` testing with a mixed-realism evidence set.

Goal:
- give the auditor enough evidence to cite specific uploaded files
- include a few genuine gaps and one mostly acceptable record
- test whether the auditor distinguishes between missing controls and acceptable basics

Suggested audit prompt:

`Audit this takeaway in England. Review the uploaded records for illness exclusion, temperature monitoring, cleaning control, and allergen readiness. Tell me the main findings, severity, immediate containment, and corrective actions.`

Included files:
- `01-scenario-note.md`
- `02-fridge-temperature-log.md`
- `03-return-to-work-form.md`
- `04-cleaning-schedule.md`
- `05-allergen-information-sheet.md`

Expected pressure points:
- fridge monitoring gaps
- one out-of-spec fridge reading
- incomplete illness exclusion / return-to-work control
- incomplete allergen information control
- cleaning schedule is present and should not be treated as the main failure

# Homepage Revamp Design

**Goal:** Refresh the homepage so it feels more credible and product-led, with consultancy acting as trust reinforcement rather than the main story.

**Direction:**
- Keep the homepage concise.
- Take inspiration from consultancy-site clarity, not their full page structure.
- Remove the current workflow-story section.
- Replace it with a tighter value section focused on what PinkPepper helps teams do.

**Structure:**
- Hero with stronger software-first headline, shorter subhead, and direct CTAs.
- Compact proof/trust strip for source grounding, exports, and specialist review.
- Concise value section showing what PinkPepper helps users do.
- Keep pricing and FAQ.

**Constraints:**
- Preserve the SEO-critical homepage phrases and links already asserted in tests.
- Avoid adding long narrative or audience-taxonomy sections.
- Keep the visual language aligned with the current premium site styling.
